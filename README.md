ポートフォリオ：料理タスク管理アプリ
構想：
・料理中の『カーナビ』を目指す。
  -献立を選択し、調理開始を押すと、優先順位を計算した上で直前の作業を大きくカードで表示。
  -画面端のナビゲーションに、2,3手先のタイトルも表示される。
  -長時間のベンチタイムやレンジ、煮詰めるといった作業の合間に、別の指示を通知する
  -各ステップ開始で1タップ、終了で1タップ。それ以上の操作はしない。
  -煮込み時間の開始を押した場合、その場でタイマーが作動する。(レンジで通知が来ると面倒だけど)タイマー終了時も通知できる。
・複数レシピ並行調理に対応させる
・実際に各ステップの消化スタンプも記録されることで、実際の調理時間の目安も共有される
・献立管理、買い物メモの補助もできるかも
・ReactNativeAppを想定。いつになることやら。
